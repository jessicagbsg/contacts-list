# Instructions

## Running the project

To run the project and use it you just have to run `npm i` and `npm run dev` for both repositories

```
cd backend
npm i
-- set DATABASE_URL="file:./dev.db" in the .env file --
npm run dev

cd ..
cd frontend
npm i
npm run dev
```
